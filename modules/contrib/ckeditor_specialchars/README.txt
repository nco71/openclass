INTRODUCTION
------------

The CKEditor Special Characters module overrides Drupal core handling of Special
Characters CKEditor plugin. It provides the ability to configure characters
available to users for insert.

 * For a full description of the module visit:
   https://www.drupal.org/project/ckeditor_specialchars

 * To submit bug reports and feature suggestions, or to track changes visit:
   https://www.drupal.org/project/issues/ckeditor_specialchars


REQUIREMENTS
------------

This module requires no modules outside of Drupal core.


INSTALLATION
------------

Install the CKEditor Special Characters module as you would normally install a
contributed Drupal module. Visit https://www.drupal.org/node/1897420 for further
information.


CONFIGURATION
-------------

    1. Navigate to Administration > Extend and enable the module.
    2. Navigate to Administration > Configuration > Content Authoring > Text
       formats and editors to configure the different text formats.
    3. In the CKEditor plugin settings vertical tab there is now a "Special
       characters" tab. Enter in the desired special characters, one per line.
       Choose if the default characters should be replaced.
    4. Save configuration.
